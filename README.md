# Mozhi-Medhai
Mozhi-medhai is a Tamil-English-Tamil speech translator which facilitates biingual communication. It utilizes advanced speech recognition and natural language processing technologies by recognizing tamil speech into tamil text then translates to english text. That text is then converted into speech and vice-versa. 

The project also emphasizes addressing social, legal and ethical comsiderations ensuring cultural sensitivity, data privacy, and compliance with relevant regulations.

By bridging language barriers, the translator aims to enhance accessibility and communication between Tamil and English speakers, promoting inclusivity and understanding in diverse communities.
